# SmartCart Customer Segmentation System
## Unsupervised Machine Learning 

A customer segmentation system that analyzes 2,240 e-commerce 
customer records to discover hidden behavioural patterns using 
unsupervised machine learning — enabling targeted marketing 
and churn identification.


##  Dataset
- 2,240 customer records, 22 features
- Includes demographics, purchase behaviour, 
  website activity, and campaign response


##  ML Pipeline

| Preprocessing | Missing values, outlier treatment |
| Feature Engineering | Encoding, scaling |
| EDA | Heatmap correlation analysis |
| Dimensionality Reduction | PCA (2D visualization) |
| Optimal K Selection | Elbow Method + Silhouette Score |
| Clustering | K-Means vs Agglomerative Hierarchical |
| Characterization | Segment profiling by income, spending, engagement |



##  Tech Stack

- Python, Jupyter Notebook
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
